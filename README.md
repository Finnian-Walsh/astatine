# Astatine

Astatine is a programming language that has a balance of abstraction and control.\
\
This repository contains the compiler implementation.

## Installation

Use cargo to install\
For the stable release:
```
cargo install astatine
```
For the latest release:
```
cargo install --git https://github.com/Finnian-Walsh/astatine.git
```

### Naming Rationale

Once upon a time, a Chemistry lesson took place in which elements were being taught. When Group 17 was discussed, the element [Astatine](https://en.wikipedia.org/wiki/Astatine) caught my attention; this element was only discovered in 1940 due to its sheer scarcity. This element is so scarce that observing it with the naked eye is simply out of the picture and its most stable isotope, astatine-210's half life of only 8.1 hours means that it doesn't accumulate naturally like other elements.\
\
Consequently, this programming language has been named in honor of the element `Astatine` due to its high scarcity. Furthermore, this language's use is likely to be extremely scarce just like the element it was named after.
